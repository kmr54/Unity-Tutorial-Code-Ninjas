    The button values are: 0 for the left button, 1 for the right button, 2 for the middle button

      GetMouseButtonDown evaluates to:
    • TRUE only when user first clicks
    • FALSE when user is still clicking
    • FALSE when user is not clicking
      GetMouseButton evaluates to:
    • TRUE only while user holding mouse button down
    • FALSE when user firsts clicks
    • FALSE when user is not clicking
    • FALSE when user releases button
